About Me

Howdy! I'm an incoming PhD student in Vanderbilt's Quantitative & Chemical Biology program who specializes in turning messy, cross-disciplinary problems into systematic, data-driven solutions. My toolkit was developed in settings that range from processing 100 + HPLC chromatograms in Denmark, where I wrote Python scripts to clean, analyze, and visualize the data, to building a full-stack lipidomics website that will allow real-time insight into lipidomics data and experiments. More recently, I designed reliability tools for a production RAG framework, experience that sharpened my machine-learning instincts and taught me where computational shortcuts break down in the real world. Now I am very excited to take the next step in pursuing graduate research, with longterm goals to progress rational enzyme and peptide design: leveraging ML models to streamline therapeutic discovery.

Projects

Lipid Lens: Full-Stack Lipidomics Visualization Awaiting Deployment

Developed a comprehensive, interactive web platform for the visualization and analysis of mouse lipidomics data. Engineered a full-stack solution using Django for the backend and JavaScript for dynamic data visualizations, complemented by a robust PostgreSQL database. Designed the system for intuitive use, enabling researchers to manage and gain real-time insights from complex lipid data without requiring technical expertise.

Rhodiola Rosea Bioactive Compound Analysis Complete

Collaborated with a cross-cultural team in Denmark to investigate the correlation between growth location, plant sex, and the concentration of bioactive compounds (rosavin and rosarin) in Rhodiola Rosea. My core contribution involved the data analysis and visualization of over 100 HPLC chromatograms, including method validation, ANOVA analysis, and post-hoc significance testing using SciPy and Matplotlib.

  • HPLC
  • SciPy
  • Matplotlib

Qualitative Evaluation of Cancer Treatments on Cell Cycle Progression Complete

Led a three-person team to investigate the impact of MLN 4924, Hydroxyurea, and an undisclosed compound-X on osteosarcoma cancer cells utilizing the FUCCI system. Employed fluorescence microscopy and Fuji software to track fluorophore expression, qualitatively assessing variations to support proposed mechanisms of action for various anti-cancer compounds and their effects on cell cycle progression.

  • FUCCI System
  • Cell Culture
  • Fuji Software
Paper
Ingredient Intel: Barcode-Scanning Nutrition App Complete

Led a two-person team to develop and deploy a barcode-scanning nutrition application. This service integrates with the USDA FoodData Central (FDC) API to instantly flag potential allergens, recalls, and health hazards based on scanned product barcodes. The application features a clean user interface built with Django and Bootstrap on the frontend, supported by a cloud-hosted PostgreSQL database on Amazon RDS for robust backend functionality and reliability.

  • Django
  • AWS RDS
  • PostgreSQL
  • USDA FDC API
